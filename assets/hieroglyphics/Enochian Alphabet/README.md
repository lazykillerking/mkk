# Enochian Alphabet

This folder contains the character set for the Enochian Alphabet.

## Important Note: W = VV

In the Enochian alphabet, the letter **W** does not have its own unique character. Instead, **W is represented as VV** (two V characters).

The files in this folder are named A through Z, but note that there is no `W.png` file. The sequence goes directly from `V.png` to `X.png` to reflect this linguistic feature of the Enochian alphabet.

When rendering text in the Enochian script, any instance of the letter "W" should be replaced with two "V" characters.
